# 📄 Django Document Management System

A secure **Document Management System** built using **Django** that enables users to register, authenticate, manage their profiles, and upload, view, edit, and delete documents. The application demonstrates secure user authentication, CRUD operations, file management, and responsive web design using Django and SQLite.

# ⚙️ Installation

## 1. Clone the Repository

```bash
git clone https://github.com/aryasajeev-git/django-project.git
```

## 2. Navigate to the Project Directory

```bash
cd django-project
```

## 3. Create a Virtual Environment

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## 4. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 5. Apply Database Migrations

```bash
python manage.py migrate
```

---

## 6. Run the Development Server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
